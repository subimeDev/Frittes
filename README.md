# Frittes

Sitio estatico del menu de Frittes Maison.

## Deploy en Railway

Este repositorio queda listo para Railway usando Docker:

- `Dockerfile` basado en `nginx:alpine`.
- `index.html` como pagina de entrada.

### Pasos en Railway

1. Crear proyecto nuevo en Railway.
2. Conectar este repositorio de GitHub.
3. Railway detecta `Dockerfile` y construye automaticamente.
4. Publicar el servicio y usar el dominio generado.
